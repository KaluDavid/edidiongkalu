## Project Overview

This website was developed to bring the client's design direction to life with clean implementation, smooth animations, and a structure that complements the storytelling-driven brand.  
The goal was to **translate her Figma vision into a high-quality, clean implementation** while maintaining clarity, consistency, and responsiveness.

---

## Live Preview

**[Live Website Link](https://edidiongkalu.vercel.app/)**  
## Tech Stack

- Next.js + TypeScript  
- Tailwind CSS  
- Framer Motion  
- Figma-to-Code Implementation 

---

## Key Features

- Full translation from **Figma to code**, preserving design intent  
- Fully responsive layout (desktop, tablet, mobile)  
- Soft, fluid animations using Framer Motion  
- Modern branding aligned with the client's personality and design tone  
- Clean component structure for scalability and future updates  
- Accessible, SEO-friendly, and performance-optimized
